## Browser Automation Setup

**Status:** ✅ Configured and live

**Configuration location:** `openclaw.json` with CDP driver

**Capabilities:**
- Chrome/Chromium control via Chrome DevTools Protocol (CDP)
- Browser profile: "openclaw" 
- Color: #FF6B00
- Email automation through Gmail interface
- Form filling and interaction
- Tab/window control

**What this enables:**
- Direct Gmail control for email composition and sending
- Form automation across web applications
- Browser tab management and content interaction
- Email drafting, subject line input, body text entry, send button control

**How to use:**
1. Ensure Chrome is running with OpenClaw profile
2. Open target application (Gmail, forms, etc.) in a browser tab
3. Use browser automation to control the interface directly
4. Supports email sending through Gmail web interface
