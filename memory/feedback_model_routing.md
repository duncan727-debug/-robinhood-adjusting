**Rule:** Always use Claude Sonnet 4.6 as the default model for all work.

**Why:** Sonnet 4.6 provides the right balance of speed and capability for the full range of work we do—from quick edits to complex automations. Haiku is too limited; Opus is overkill and slower.

**How to apply:** When starting a new task or session, verify the active model is Sonnet 4.6. If you're in a session with a different default (Haiku, Opus, etc.), ask Duncan or switch to Sonnet 4.6.
