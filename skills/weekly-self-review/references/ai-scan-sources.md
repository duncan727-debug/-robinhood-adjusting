# AI / tooling scan sources

Update this list quarterly.

## Primary
- anthropic.com/news — Claude model + SDK releases
- openai.com/blog — GPT, agents, file API
- x.com/AnthropicAI, x.com/OpenAIDevs — fast signal
- github.com/anthropics — SDK changelogs, claude-code releases
- github.com/openclaw — OpenClaw harness releases
- modelcontextprotocol.io — MCP spec + new servers

## Secondary
- huggingface.co/blog — OSS models worth noting
- LangChain, LlamaIndex, CrewAI release notes — agentic frameworks
- Latent Space podcast / Swyx feed
- Dwarkesh interviews — capability frontier
- ai-engineering newsletters: Latent Space, AI Engineer Summit talks

## What to surface (1-liner verdicts)
For each item, answer:
1. What changed?
2. Could Duncan's ops benefit? (specific scenario)
3. Effort to adopt? (S/M/L)
4. Risk if we don't? (low/med/high)

Skip anything that doesn't have a plausible answer to #2.
