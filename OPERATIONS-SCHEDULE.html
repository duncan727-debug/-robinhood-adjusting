How It Works

  • Morning Block (4am–7:30am EDT): All prep work complete before 8am — brief, content, intelligence, outreach, responses, review
  • Input → Output Chain: Brief (4am) → Content (4:30am) → Intelligence (5am) → Outreach (6am) → Responses (7am) → Review (7:30am)
  • All outputs are drafts: Nothing posts or sends automatically. Duncan reviews and approves manually
  • Auto-Recovery: LaunchAgent detects Mac wake-up, triggers any missed jobs in correct order. State file prevents duplicate runs same day
  • 1-Hour Time Slots: Tasks now align consistently across all days — BRIEF always at 4am position, OUTREACH always at 6am position, etc.
  • Color Coding: Teal = Newsletter work | Purple = CRM/Prospecting | Gray = Operations | Red = Monthly reviews
  • Mon-Sat full operations: Morning block runs Mon–Sat. Sunday is rest day.

Auto-Recovery / Catchup System (Active)

  • LaunchAgent: com.openclaw.catchup runs every 5 minutes at launch/boot
  • Catchup Script: /Users/victoria/.openclaw/catchup.sh detects time and which jobs should have run
  • State Tracking: .catchup-state.json prevents duplicate runs same day
  • Logging: All activity logged to .catchup-log.txt
  • Job Order: Runs in dependency order (brief → content → intelligence → outreach)
  • When it triggers: Mac wakes up, or next time LaunchAgent checks (every 5min) if jobs missed
