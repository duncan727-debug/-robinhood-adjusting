## Pro Tips for LinkedIn Outreach

1. **Connection Request Notes:** LinkedIn allows personalized text in connection requests—the messages in each sequence file are optimized for this. Use them.

2. **If Connection Notes Don't Work:** Send standard connection request, then immediately DM the same message. LinkedIn often lets you DM non-connections.

3. **DM Timing:** Send DMs during business hours (9-11 AM or 3-5 PM EDT). Weekday DMs get faster responses.

4. **DM Length:** These messages are conversational but concise—LinkedIn users prefer short DMs over long walls of text.

5. **Call Requests:** In STEP 3, you're asking for 15-20 minute calls. These should be follow-up *only*—don't ask for calls in initial connection requests.

6. **Personalization:** Each sequence uses specific details from the intelligence (veteran background, Category 5 expertise, capacity constraints, etc.). This specificity *matters*—it shows research, not generic outreach.

7. **Warm Reactivation (KLR):** You've already emailed KLR three times. LinkedIn is a fresh channel that may get a response when email hasn't. Use it as reactivation, not redundant outreach.

## Next Steps After They Accept

4. **Follow up via email with call notes** (confirm what you discussed, next steps in writing)
