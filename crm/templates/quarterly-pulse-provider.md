# Template: Quarterly partner pulse

**Use when:** 90 days have passed since previous pulse (or 90 days since 30-day check-in for fresh partners). Automated by `partner_lifecycle.py`.

**Format rule:** Network update + transparency. Treats partner like an insider. Keeps the directory feeling alive even when no referrals have flowed.

**Parameters to fill in:**
- `{FIRSTNAME}` — partner first name
- `{COMPANY}` — business name
- `{REFERRAL_COUNT_QUARTER}` — referrals sent to them this quarter (0 if none)
- `{NETWORK_SIZE}` — current verified provider count (e.g. "67")
- `{NEW_PARTNERS_THIS_QUARTER}` — number of new verified providers added this quarter
- `{NETWORK_HIGHLIGHTS}` — 2–3 bullets of recent network/market context (storm activity, policy changes, notable additions)
